## Контрольные вопросы
1. Типы данных SQLite**: NULL, INTEGER, REAL, TEXT, BLOB. Логические значения хранят как INTEGER (0/1), даты — как INTEGER (Unix timestamp) или TEXT (ISO 8601).

2. SQLiteOpenHelper** — класс для управления созданием и обновлением БД. `onCreate()` вызывается при первом создании БД, `onUpgrade()` — при изменении версии для обновления структуры.

3. getWritableDatabase()** возвращает БД для чтения и записи, **getReadableDatabase()** — для чтения (может вернуть writable при доступности). Ошибка `getWritableDatabase()` возникает при нехватке места или если БД только для чтения.

4. Cursor** — указатель на результат запроса. Перемещаются методами `moveToFirst()`, `moveToNext()`, `moveToPosition()`. Закрывать обязательно через `close()` или try-with-resources для избежания утечек памяти.

5. ContentValues** — контейнер для пар "столбец-значение", используется в `insert()` и `update()`.

6. query()** — высокоуровневый метод с параметрами, **rawQuery()** — принимает чистый SQL. Пример: `db.rawQuery("SELECT * FROM trains WHERE id=?", new String[]{"5"})`.

При изменении структуры таблицы увеличивают версию БД и в `onUpgrade()` используют `ALTER TABLE ADD COLUMN` для добавления полей или создают временную таблицу для сложных миграций.
